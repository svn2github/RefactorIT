Move Method/Field

This function moves a method or field declared in class to another class or interface.

Usage



Choose field or method you want to move and the new class or interface it should belong to. All references in classes are updated where necessary and if the move causes conflicts they will be shown in lower part of the window. They must be resolved in order to perform refactoring.
