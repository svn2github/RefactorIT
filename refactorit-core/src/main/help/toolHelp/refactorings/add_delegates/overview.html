Add Delegate Methods

Creates delegate methods to the field of the target class. Delegate methods are methods invocating selected target class fields' methods.

Usage

To execute this action, click on class or class field name and select 'Add Delegate Methods' from RefactorIT menu. The following window will be opened for you:


     

From the dialog you can select methods to create for the field by expanding class tree branches and ticking checkboxes corresponding to potential delegate methods.


     

When all necessary methods are selected click ok. As a result you will be presented with a window displaying information about the changes that are going to be performed by RefactorIT:


     

Apply any corrections if needed and click ok. Changes will be performed.


Results

If delegate method operation was not canceled by user and no errors occured the new field delegate method will be added to field owner class body.
