Command Line Interface

The Audit, Metrics and Not Used operations can be ran from the command line and by a custom Ant task. Results can be saved to files in various formats, so they can be analysed by spreadsheet tools, or, for example, sent via e-mail to the development team as part of the nightly build.

Results with too much "noise" will tend to be skipped by people, which is why Audit and Metrics results are optionally configurable using profiles.
