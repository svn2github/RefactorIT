RefactorIT Ant task

Description

Runs RefactorIT actions, such as audit, metrics and not used. For a quick overview, see a working sample file.

Installation

Just install RefactorIT in stand alone mode (it is an installer option). In build files, use the following task definition (be sure to replace "C:\RefactorIT" with your RefactorIT home folder):

<taskdef name="RefactorIT" classname="net.sf.refactorit.ant.RefactorIt">
  <classpath>
    <fileset dir="C:\RefactorIT" includes="**/*.jar"/>
  </classpath>
</taskdef>
