Where Used For Fields

Usage

Usages of a field are:

  • Setting the value of the field, except during declaration.

  • Getting the value of the field.

You can tune the results of the query in the filter dialog for variables that automatically open when you run the query.

Example

class Test {
  String text = "Hi!"; // Not counted as usage
                       // (declaration)

  void test() {
    text = "Hello, World!";            // set
    System.out.println(text.length()); // get
    System.out.println(text);          // get
  }
}
