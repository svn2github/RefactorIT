Where Used For Methods

Usage

Usages of a method are:

  • Invocation of the method.

  • Declaration of overriding/implementing method in subclass.

You can tune the results of the query in a filter dialog for methods. One such dialog is for private methods and constructors and another one is for non-private methods. The dialog will be opened automatically when you run the search.

Example


class A {
  void theMethod(String tmp) {}

  void test() {
    theMethod("Hello"); // invocation
  }
}

class B extends A {
  void theMethod(String a) { // overriding
  }                          // A.theMethod

  void test(A a) {
    a.theMethod(null); // invocation
  }
}
