Filters

You can narrow down the results of a Where Used query by choosing different settings from the query's filter dialog.

Usage

The filter dialog comes up automatically every time you run a Where Used query. You can also open that dialog later from the results panel to refine the search by clicking on the filter button after you have obtained initial results.

References

RefactorIT has different specialised filters for different queries:

  • Filter for class usages

  • Filter for interface usages

  • Filter for usages of non-private methods

  • Filter for packages usages

  • Filter for variable usages

  • Filter for constructor and private method usages
