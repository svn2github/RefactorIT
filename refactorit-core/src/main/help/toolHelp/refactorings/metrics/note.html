Number of Exported Types (NOTe)

NOTe reports the number of classes and interfaces that are exported outside of a package. The default limit for NOTe in RefactorIT are 3 and 50. A large value of NOTe points to an extensive coupling for which reason this should be considered as a value to minimize.
