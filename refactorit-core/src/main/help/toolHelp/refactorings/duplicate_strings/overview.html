Find Duplicate Strings

Usage

This feature allow you to quickly find duplicated strings in your code. This information may be used in different ways. Most common reason for this - the need in locating all duplicated strings, introduce a new constant containing this string and replace all string occurrences with this constant call. This helps to change/fix/translate specified strings, since all are concentrated in one place, what is definitely better for control purpose and, therefore, for you code design.

Example

public class DuplicateStrings {

  public void printFirstItem() {
    System.out.println("Printing item: " + "item1");
  }
  
  public void printSecondItem() {
    System.out.println("Printing item: " + "item2");
  }
  
  public void printThirdItem() {
    System.out.println("Printing item: " + "item3");
  }
}

To use this feature - simply select package or class definition and run this action. Results window will show you all duplicated strings. After studying this information you may decide to introduce a new field constant using Introduce Explaining Variable and Convert Temp To Field refactorings like this: 

public class DuplicateStrings {
  public static final String PRINTING_ITEM_ACTION = "Printing item: ";

  public void printFirstItem() {
    System.out.println(PRINTING_ITEM_ACTION + "item1");
  }
  
  public void printSecondItem() {
    System.out.println(PRINTING_ITEM_ACTION + "item2");
  }
  
  public void printThirdItem() {
    System.out.println(PRINTING_ITEM_ACTION + "item3");
  }
}

Results

The results of this action is a list of duplicated strings. It may be used for improving performance and reducing resource allocation or for the common data concentration.
