Call Tree

Shows all possible stack traces for a given method, constructor, or field.

Usage

Choose the target and select 'Call tree' command from RefactorIT menu.

Results

The results appear as an exhaustive recursive "Where Used" ran in an 'implementation' mode until all occurrences are found.
Note: It can be useful to observe the way some fields can be accessed or changed during the runtime of the program.
