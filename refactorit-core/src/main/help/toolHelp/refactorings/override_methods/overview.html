Override/Implement Methods

This refactoring creates desired method overrides of superclass methods chosen from list of potential method overrides for current class

Usage

Use this refactoring to override superclass methods by clicking on target class. Override/Implement dialog will appear:


     

Dialog window will offer You to choose superclass methods, that are not already overridden in current class. Select methods You would like to override by ticking appropriate checkboxes and press 'ok'.

Changes preview dialogwill appear offering You to view and edit changes RefactorIT is going to perform (method equals() was selected in current example):


     

Click 'ok' when done.

Example

In this example such method will be added into class body

    public boolean equals(Object obj) {
      
      // @todo: override this 
      return super.equals(obj);
    }

Results

Overridden methods` code will be added to destination class source code as it was shown in changes preview dialog. Method bodies will contain superclass method calls by default.
