Unicode Support

RefactorIT has Unicode support. You might need to change character encoding in the RefactorIT Options dialog, because RefactorIT does not import source encoding settings from the IDEs it runs under.

NOTE: RefactorIT does not support unicode escape characters outside string or a character literals. The limitation is a price to be paid for the speed of the RefactorIT.

Please write to support@refactorit.com if you find any problems with special characters.
