Where Caught

Usage

Finds where an exception is handled and is invoked when clicked:

  • on any exception name declared in the throws clause of the method or constructor declaration, e.g. when clicked on MyException string in

    
  public void method() throws MyException, SomeOtherException {
    ..
  }

  • on the throw keyword within the body of the method or constructor, e.g.

    
  throw new MyException(); // click within 'throw'

Results

The result is a tree of all the method/constructor invocation paths an exception can fly through until caught.
