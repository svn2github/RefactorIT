Move

Moves chosen element to desired destination. Can be applied on class, interface, method and field.


Usage

To use this action choose element to move (class, interface, method ot field) and click on element`s name. Choose 'Move' item from RefactorIT menu.

Class/Interface Move

When applied on class or interface, move dialog is executed. Dialog offers to move chosen class or interface to other package. See Move Class section for details.


Method/Field Move

When applied on method or field, method/field move dialog is executed. Dialog offers selection of possible destinations for chosen element and information about possible move conflicts. See Move Method/Field section for details.


Results

As a result - the desired move refactoring will be fulfilled and chosen element will be moved to selected destination.


References

  • Move Class - applies to classes and interfaces

  • Move Method/Field - applies to methods and fields
