Redundant unboxing

Searches the code for redundant unboxing.

Example:


    public void a(){
        Integer foo = 5;
        int bar = foo.intValue(); // redundant unboxing
    }

Here, calling method Integer.intValue() is not necessary, because in J2SE5 Integer is unboxed to int automatically.


Corrective actions:

Remove redundant unboxing

Example:

before:
    Integer foo = 5;
    int bar = foo.intValue();
after:
    Integer foo = 5;
    int bar = foo;
