Unused Local Vars

This warning reports unused local variables, optionally including method and catch clause parameters.


Corrective actions:

Remove unused local variable

Action removes unused local variable declaration.

Example:

before:
        void doSomething(){
            int a = 5;
            for (int i = 0; i < 10; i++){
                doSomethingWith(i);
            }
        }
after:
        void doSomething(){
            for (int i = 0; i < 10; i++){
                doSomethingWith(i);
            }
        }

Remove parameter from method's hierarchy signatures

Action removes unused parameter from all method inheritance hierarchy.

Example:

before:
        class A {
            public void method(String s, int i){
                System.out.println(s);
            }
        }
        
        class B extends A {
            public void method(String s, int i){
                System.out.println(s);
            }
        }
after:
        class A {
            public void method(String s){
                System.out.println(s);
            }
        }
        
        class B extends A {
            public void method(String s){
                System.out.println(s);
            }
        }
