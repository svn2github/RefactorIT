Dangerous catch clauses

Finds catch clauses with formal parameters of types java.lang.Error, java.lang.Throwable or any of their subclasses, except java.lang.Exception.

Example 1: 

    try {
        ...
    } catch (Throwable e){} // violation!

Example 2: 

    try {
        ...
    } catch (Error e){} // violation!

Example 3: 

    class MyThrowable extends Throwable {}
    try {
        ...
    } catch(MyThrowable e){} // violation!

Example 4: 

    try {
        ...
    } catch(Exception e){} // no violation
