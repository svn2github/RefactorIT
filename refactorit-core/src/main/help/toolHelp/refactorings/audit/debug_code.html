Debug code

This warning reports following cases of printing debug information to console:

System.out.println("any text here");
System.err.println("any text here");
exception.printStackTrace();


Corrective actions:

Remove debug code

Action removes debug code.

Example:

before:
    int function(){
        int value = getSomeValue();
        System.out.println("value = " + value);
        return value;
    }
after:
    int function(){
        int value = getSomeValue();
        return value;
    }

Comment out debug code

Action comments out debug code.

Example:

before:
    int function(){
        int value = getSomeValue();
        System.out.println("value = " + value);
        return value;
    }
after:
    int function(){
        int value = getSomeValue();
    //    System.out.println("value = " + value);
        return value;
    }
