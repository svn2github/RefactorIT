Staticalizable methods

This audit finds all methods, that do not use any non-static members of their owner types and proposed to make them 'static'.

Example:

    class A {
        int a = 5;
        
        public void printA(){ // no violation
            System.out.println("a = " + a);
        }
        
        public void printTest(){ // violation: can be 'static'
            System.out.println("2 + 2 = " + (2 + 2));
        }
    }

Corrective actions:

Accept suggested 'static' modifiers

Adds 'static' modifier to method declaration.

Example:

before:
    class A {
        public void printTest(){
            System.out.println("2 + 2 = " + (2 + 2));
        }
    }
after:
    class A {
        public static void printTest(){
            System.out.println("2 + 2 = " + (2 + 2));
        }
    }
