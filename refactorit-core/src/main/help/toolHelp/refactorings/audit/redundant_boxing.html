Redundant boxing

Searches the code for redundant boxing.

Example:


    public void a(){
        Integer foo = new Integer(5); // redundant boxing
    }

You do not need to create new object of type Integer in J2SE5: int is automatically boxed to Integer.


Corrective actions:

Remove redundant boxing

Example:

before:
    Integer foo = new Integer(5);
after:
    Integer foo = 5;
