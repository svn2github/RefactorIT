Not serializable superclass

A class implementing java.io.Serializable should have serializabe superclass, or superclass with an accessable default (no argument) constructor.

Check java.io.Serializabe API reference for details.


Corrective actions:

Make superclass java.io.Serializable

Example:

before:
    public class A {
      ...  
    }
    
    public class B extends A 
        implements java.io.Serializable {
      ...
    }
after:
    public class A implements java.io.Serializable{
      ...  
    }
    
    public class B extends A
        implements java.io.Serializable {
      ...
    }

Create accessable no-argument constructor for superclass

Example:

before:
    public class A {
      ...  
    }
    
    public class B extends A
        implements java.io.Serializable {
      ...
    }
after:
    public class A {
      public A(){
      }
      ...  
    }
    
    public class B extends A
        implements java.io.Serializable {
      ...
    }
