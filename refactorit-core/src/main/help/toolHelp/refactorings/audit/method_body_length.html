Method body length

This audit rule finds too short and too long methods.

It reports all methods, for which is true, that L < n < U, where:
L - given lower bound (default is 2),
U - given upper bound (default is 20),
n - number of statements in method body.

Pay attention, that this audit doesn`t count *lines* of code, but *statements*.

In audit options You can set lower/upper bound values. 'Skip getter/setter methods' option allows to skip violations, about too short getXXX() and setXXX() methods.
