Finalizable types and methods

This audit finds all methods and classes, which could be made 'final'.


Corrective actions:

Accept 'final' modifier for classes

Adds 'final' modifier to type declaration.

Example:

before:
    public class A {
    }
after:
    public final class A {
    }

Accept 'final' modifier for methods

Adds 'final' modifier to method declaration.

Example:

before:
    public class A {
        public void method(){
        }
    }
after:
    public class A {
        public final void method(){
        }
    }
