Generics usage

This audit rule finds all variables of generic types, which are used without type arguments.

Example:


    List list = new ArrayList(); // violation
    list.put("string");
    String element = (String) list.get(0);

List is generic type in J2SE5, and in this the code could look like this:


    List<String> list = new ArrayList<String>();
    list.put("string");
    String element = list.get(0);
