Boolean assignment in loop conditions

Searches the code for boolean assignments in loop condition expressions.


    boolean a = false;
    boolean c = true;
    .......
    while (a = c){ // loop forever?
        ...
    }

Such bugs are often hard to find and compiler doesn`t report any error here.


Corrective actions:

Replace assignment with comparison operator

This action replaces '=' operator with '==' operator, turning assingment expression into logical expression.

Example:

before:
    while (a = c){
    }
after:
    while (a == c){
    }
