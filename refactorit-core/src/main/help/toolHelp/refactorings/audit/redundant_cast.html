Redundant casts expressions

This audit finds all cast expressions, which could be safely omitted.

Example:


    public void parse(List listOfStrings){
        // why cast to String if assigning to variable of type Object?
        Object obj = (String) listOfStrings.get(0);
    }

There are many other places, where explicit cast is redundant (audit also supports J2SE5 Generics/Boxing features).


Corrective actions:

Remove redundant casts

Example:

before:
    Object obj = (String) listOfStrings.get(0);
after:
    Object obj = listOfStrings.get(0);
