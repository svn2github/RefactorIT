String.toString() conversions

This audit searches the code for String.toString() conversions. There is no sense invoking toString() method on a String, because String.toString() just returns the same string object, on which it was invoked.


Corrective actions:

Remove redundant toString() call

Example:

before:
    String s = "abc";
    System.out.println(s.toString());
after:
    System.out.println(s);
