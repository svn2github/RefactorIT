Empty statement

This warning reports empty statements

For example:

    int a = 5;
    ; // redundant empty statement
    a++;; // redundant empty statement (second semicolon)

Corrective actions:

Remove redundant semicolon

Action removes redundant semicolon.

Example:

before:
    int a = 5;
    ;
    a++;;
after:
    int a = 5;
    a++;
