Not serializable superclass

java.io.Serializable class should have serialVersionUID member field with following signature:

<any-access-modifier> static final long serialVersionUID = ...;

Read more

This audit rule reports all java.io.Serializable classes without properly declared 'serialVersionUID'.


Corrective actions:

Add serialVersionUID member field

Example:

before:
    public class A implements java.io.Serializable {
      ...  
    }
after:
    public class A implements java.io.Serializable {
      public static final long serialVersionUID = ...L;
      ...  
    }

Correct serialVersionUID modifiers

Example:

before:
    public class A implements java.io.Serializable {
      public final int serialVersionUID = 555;
      ...  
    }
after:
    public class A implements java.io.Serializable {
      public static final long serialVersionUID = 555;
      ...  
    }
