Empty blocks

This warning reports missing blocks for 'if', 'while', 'for' and 'do' statements:

For example:

if(something){

}

This is possibly an unimplemented part of code. To suppress this warning, you can either disable it from the filter dialog or write some code or comments inside.


Corrective actions:

Mark with CAUTION comment

Action adds /* CAUTION: empty block or body */ comment, so this place could be found afterwards with Fixme Scan.

Example:

before:
    if(something){

    }
after:
    if(something){
        /* CAUTION: empty block or body */
    }
