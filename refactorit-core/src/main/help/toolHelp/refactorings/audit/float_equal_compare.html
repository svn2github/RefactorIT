Floating point variables equal comparision

This warning reports cases of comparing floating point variables with '==' or '!=' operators.

Example: 

    float pi = 3.14;
    if (pi == Math.PI){ // violation
        // never gets here
    }

You should avoid comparing floating point variables this way. The correct way to compare floating point variables would be compare them with given precision: 

    if (Math.abs(pi - Math.PI) < 0.01){
        // gets here now
    }


Corrective actions:

Change to 'Math.abs(a - b) < precision'

This action will show a dialog, where You can enter desired precision, and then will replace 'a == b' expression with 'Math.abs(a - b) > precision' expression.

Example:

before:
    if (pi == Math.PI){
    }
after:
    if (Math.abs(pi - Math.PI) < 0.01){
    }
