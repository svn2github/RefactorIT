Dangerous throw statements

Finds throw clauses that declare java.lang.Error, java.lang.Throwable or any of their subclasses, except java.lang.Exception.

Example 1: 

    public void m() throws Throwable){ // violation!
        throw new Throwable();
    }

Example 2: 

    public void m() throws Error){ // violation!
        throw new Error();
    }

Example 3: 

    class MyThrowable extends Throwable {}
    
    public void m() throws MyThrowable){ // violation!
        throw new MyThrowable();
    }

Example 4: 

    public void m() throws Exception){ // no violation
        throw new Exception();
    }
