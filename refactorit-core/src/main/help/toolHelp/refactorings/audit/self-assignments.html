Self-assignments

This warning reports assignments for which the left and right side are the same:

    ...
    int x;
    boolean b;
    ...
    x = x; // redundant, possible bug
    ...
    if(b = b) // redundant, possible bug
    ...

Corrective actions:

Remove self-assignment

Action removes self-assignment expression.

Example:

before:
    int x = 0;
    ...
    x = x;
    ...
after:
    int x = 0;
    ...
    ...
