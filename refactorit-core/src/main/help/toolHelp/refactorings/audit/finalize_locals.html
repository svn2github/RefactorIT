Finalizable local variables and parameters

This audit finds all local variables and method parameters, which are assigned only once during lifetime. Their variable declarations could safely be made 'final'.


Corrective actions:

Accept 'final' modifier for local variables

Adds 'final' modifier to local variable declaration.

Example:

before:
    public class A {
        public int method(){
            int i = 0;
            System.out.println("i = " + i);
            return i;
        }
    }
after:
    public class A {
        public int method(){
            final int i = 0;
            System.out.println("i = " + i);
            return i;
        }
    }

Accept 'final' modifier for method parameters

Adds 'final' modifier to parameter declaration.

Example:

before:
    public class A {
        public int echo(int i){
            System.out.println("i = " + i);
            return i;
        }
    }
after:
    public class A {
        public int echo(final int i){
            System.out.println("i = " + i);
            return i;
        }
    }
