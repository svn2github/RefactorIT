Change Method Signature

Changes method signature according to user requirements. Allows changes of method return type, modifier, number of parameters, their order, types, names and default values. Changes are performed for all method usages in source code including the method itself.

Usage

To execute this refactoring, click on class or method`s name and select 'Cnahge Method Signature' from RefactorIT menu. The following dialog will appear:


     

Method Modifier

To change method modifier, select appropriate modifier type from the dialog field titled with 'Modifier'.


Return type

Click selection button '...' located on the right of return type field. The return type selection dialog will appear.


     

  • You can search type by typing its name into text field on dialog`s 'Search Class' tab. See image

  • Return type can also be selected from the tree of project classes on dialog`s 'Browse Class' tab. See image

  • Primitive types are located in 'Browse Class' tree`s bottom branch. See image

When required return type is located and selected, click ok. Return type selection dialog will close and new return type will appear in 'Return type' field of 'Change Method Signature' dialog.

Parameters

  • Type

    - To change parameter type click '...' button located in parameter type field of the dialog. The type selection dialog will appear. Select required type and click 'ok'.

    • - Paramater type can also be changed by double clicking on 'Type' field. The field becomes editable.

  • Name

    - To change parameter name double-click on 'Name' field. The field becomes editable.

  • Position

    - To change parameter order select parameter by clicking on a corresponding row in parameter table. To select several parameters hold 'Control' button. Selected rows become highlighted. Now You can move selection`s relative position using 'Up' and 'Down' buttons.

  • Remove

    - To remove one or several parameters select appropriate rows in parameter table and click 'Remove' button.

  • Add

    - To add a parameter click 'Add' button. Parameter type selection dialog will appear. Select appropriate parameter and click 'Ok'. If selection was made correctly, parameter name dialog will appear. Input new parameter name and click 'ok'. New parameter row should appear in parameter table.

Preview

You can see how changed method signature will look like in 'Method signature preview' field of 'Change Method Signature' dialog.

Results

If method owner already contains a method with the same signature as defined during signature change dialog, You will be prompted to rename current method to avoid duplicate method appearance. If no errors appear old method signature will be changed into user defined signature, considering all method usages in source code and parameter usages in method body.
