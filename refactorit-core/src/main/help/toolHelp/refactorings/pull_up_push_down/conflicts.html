Conflicts

The moving of fields or methods may cause conflicts which are displayed in lower part of the window. Conflict can be:

Resolvable

If marked by   icon that means that the only resolution exists for this conflict. Select conflict and press 'Resolve Conflict' button to resolve it.

If marked by   icon that means that a few resolutions exist for this conflict. Select conflict, press 'Resolve Conflict' button and choose from the list of possible resolutions.

Unresolvable

Is marked by   icon. These conflict cannot be resolved by RefactorIT and if such conflict emerges, refactoring cannot be continued.
