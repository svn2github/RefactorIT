Clean Imports

Clean Imports removes unnecessary import statements.

Usage

To execute this action, simply right click on java file, package, project or class and choose "Clean imports" in the opened window. Window with impotrs to be removed will appear. If there are any imports you do you do not want to remove, deselect them, and then press OK.

Example

	import java.util.HashSet;
	import java.util.ArrayList 
		
	public Interface SomeInterface {
		ArrayList list1;
		ArrayList list2;
	}

Results

	import java.util.ArrayList; 
		
	public Interface SomeInterface {
		ArrayList list1;
		ArrayList list2;
	}

Import of java.util.HashSet was removed, as it is never used.
