Inline

The "Inline Method" places the method's body into the body of its callers

Usage

You can run Inline Method on a name in method declaration or on a name in method invocation. It applies to methods, local variables and fields.

"Inline Variable" replaces all references to a variable with the expression in its declaration and removes the declaration. Inline variable can be ran on fields with some restrictions.
