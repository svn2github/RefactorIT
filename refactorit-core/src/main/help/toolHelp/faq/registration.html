Registration - Can not connect to RefactorIT server

Sometimes registration process fails because of a firewall or some other network problems. Try to find the correct firewall settings for your network and enter it in the appropriate page in the RefactorIT registration dialog.

Registering via browser

If nothing helps it is still possible to register through the browser. The template of the address is: http://activation.refactorit.com/ipay/keychecker/? site_user=&pass=&key=&system_user= &send_license=1&send_method=direct

You just need to fill in the missing details (after '=' signs):

  1. site_user - your account name on our site

  2. pass - the password on the site

  3. key - activation key you should have received by the mail

  4. system_user - login name on your operating system, the license will work with this user only (case sensitive)

If everything is correct you will a receive binary stream that must be saved into a file refactorit.license and can be imported into RefactorIT with the 'Activate...' button in the About RefactorIT dialog.

Support contact

In case of a problem with registering through the browser you will receive a page with a single number in the upper-left corner and must contact us at support@refactorit.com.

Alternatively you can just send us your system user name (login) mentioned above and we generate the license for you.
