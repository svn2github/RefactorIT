Troubleshooting and Tips

Below you will find a list of Troubleshooting items. If you can't find an answer for your problem from here then please visit our website www.refactorit.com. For support questions you can either post your question to our forum or e-mail to support@refactorit.com

If RefactorIT shows parsing or resolving errors on the project errors tab and you are sure that your code compiles correctly, then:

  • Is project sourcepath correct?

  • Does project classpath contain all imports you need?

If you are sure that both are correct, then:

  1. Choose Rebuild or Clean command from RefactorIT menu. This forces RefactorIT to rebuild project.

  2. Restart IDE
