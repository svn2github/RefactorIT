Thread dump

Thread dumps are most valuable in cases of hangs. Simply send them to support@refactorit.com or file a bug report.

How to get a thread dump

Under Windows: Press Ctrl-Break in the IDE console window. (To make that console window visible, start your IDE from runide.exe/JBuilder.exe/jdev.exe instead of runinde*w*.exe/JBuilder*W*.exe/jdev*w*.exe.)

Under UNIX: Send a kill -QUIT signal to the IDE java process (not the main IDE process)

Check also <user_home>/.refactorit/refactorit.log file for the presence of any stack traces and error messages.
