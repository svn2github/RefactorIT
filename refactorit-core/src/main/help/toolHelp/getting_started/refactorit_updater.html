RefactorIT Updater

It is possible for RefactorIT to renew itself when a new version has become available. This feature is called RefactorIT Updater and can be run from your IDE. To find the updater go to RefactorIT menu.

Exact menu path to the RefactorIT Updater

  • JBuilder: RefactorIT->RefactorIT Updater
  • JDeveloper: RefactorIT->RefactorIT Updater
  • Netbeans: RefactorIT->Help&Settings->RefactorIT Updater
  • Standalone: Help->Update
  • Eclipse: Eclipse owners shall use Eclipse native Update Center, that is located at Help->Software Updates->Manage Configuration.

In order to update RefactorIT you need to select RefactorIT Updater from the RefactorIT menu and follow the instructions. Click Finish when you are done. RefactorIT Updater will load the new version of RefactorIT from the update site and install it. Restart for your IDE is needed in order for the changes to come into effect.
