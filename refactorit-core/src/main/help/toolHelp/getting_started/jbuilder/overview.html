Integration with JBuilder

RefactorIT integrates with JBuilder starting with version 5. To check if RefactorIT has been successfully integrated, look for RefactorIT in the main menu. The RefactorIT menu should contain several submenu items. In addition, you should see a number of RefactorIT buttons in the main toolbar. RefactorIT takes its source path and classpath from the JBuilder Project Settings. Click the link for more detailed information on RefactorIT specific project settings.

Productivity! Compatibility Mode for RefactorIT

RefactorIT has Productivity! compatibility mode which could be useful if you have Productivity! and RefactorIT integrated on the same JBuilder installation.

Productivity! is a JBuilder plugin from AMIS Software that can interfere with RefactorIT. The usual clash results in an inaccessible functionality error, refer to RefactorIT Bugzilla . When RefactorIT is run in a compatibility mode (which can be turned on/off from the RefactorIT tab in the JBuilder Project Properties dialog), there will be less interference at the expense of removing RefactorIT menu items from the JBuilder structure view. Note: Restart JBuilder for the settings to take effect.
