Integration with SunTM ONE Studio and NetBeans

To verify that RefactorIT has successfully integrated with SunTM ONE Studio/NetBeans, open the Tools menu and look for the RefactorIT menu item with submenus.

Following the project definition guidelines of Sun ONE Studio/NetBeans, RefactorIT should be able to detect its project settings automatically, but if this does not work, please manually adjust the settings.

A recent upgrade has been made to RefactorIT where if you "Rename" a class that is directly linked in the project tab, the link will not be renamed. This is a minor change of behaviour and please inform us if you feel that it should be reversed.
