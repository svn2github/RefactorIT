Metrics

HTML report

This kind of report contains two distinct parts:

The first one is the list of all metrics/metric options used for the report generation. Notice, that 0.0 ... 0.0 means that there were no constraints defined for this metric.

The first one is the metrics characteristics list. This part contains java code element name, type and all metrics described above. If you need some more information on metric acronym - more mouse over the text, hold for some second and get the metric full name.
