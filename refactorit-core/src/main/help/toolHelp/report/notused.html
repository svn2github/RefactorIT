Not Used

HTML report

This kind of report contains three distinct parts:

  • Not used statistics. This part shows the summary information about not used java elements in the source code.
  • Not used settings. While generating the report it is possible to used exclude filter rules. Those rules are listed here for reminding.
  • Not used results. Contains detailed information on not used java elements. This includes package name, class name, member name and java element type.
