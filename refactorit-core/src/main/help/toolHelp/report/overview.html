Report Generation

RefactorIT allows to export results that were received from running Audit, Metrics or Not Used operations. Results may be presented in different formats. There is 4 formats available so far: XML, HTML, CSV and Plain Text. Generated report is possible to use for a source code and it's evolution analysis.

Warning!

Report Generation based on a large results set is a not an instant process. Usually, it takes some time to process the report, so RefactorIT may freeze for a while. The larger results set - the larger is the waiting period. To avoid this - try to be the maximum specific while you are in the operation settings dialog.
